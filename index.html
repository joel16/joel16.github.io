Cyano PSP Beta

Currently the latest stable release is Beta 2.4, however I've been slowly releasing test builds (experimentals) for v3.0. You can find more information about this program here. You can download the latest test build (Test B1 - 1/28/15) here. Information about the test builds will be posted here only.

Changes so far for v3.0

- Fixed memory leaks in both Gallery and Apollo.
- Completely Fixed back button misbehaviour on Apollo. (was being
experimented in the previous release)
- Allow access to view mp3 files in PSP/MUSIC and CyanogenMod/downloads
for Apollo.
- Minor cleaning up in gallery app.
- Fixed many unnoticed memory leaks in recovery menu.
- Fix power menu crash, when pressing circle twice.
- Display what is being done in certain options. Display whether it was a
success or not.
- Fix up last two pages of system information in the recovery menu.
- Fully Implemented brightness slider.
- Fixed lagg on welcome screen (first time boot)
- Unlock extra 4MB ram 
- Add recursive deletion in file manager.
- Possible fix for UMD loader.
- Added new menu, "Dumping Tools" under developer options. (Lets you dump UMD boot.bin, flash 0, flash 1 and memory)
